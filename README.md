# StudyHub

StudyHub er en webapplikation til at organisere noter, emner og fag.

Projektet er lavet som en del af H5 minisvendeprøven.

---

## 🚀 Tech Stack

- Next.js

- NextAuth (Google login)

- PostgreSQL (Neon)

- Drizzle ORM

- HocusPocus/Yjs (websocket server)

---
